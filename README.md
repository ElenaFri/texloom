# texloom

A simple Markdown-to-LaTeX converter and text editor. With (useful) GUI.

## Build

Compile the application with CMake:

```bash
cd app
mkdir build && cd build
cmake ..
cmake --build .
```

## Run

Launch the application:

```bash
# From app/build/
./bin/texloom
```

## Tests

Run the test suite:

```bash
# From app/build/
ctest
```
